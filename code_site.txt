site avec les boutons : register / login (login.html) -> nous envoie sur site.html si le compte est valide (voir code register.php)

sur site.html, si on est un employé on peut ajouter une empreinte si elle est pas déjà enregistrée et si on est le patron on peut ajouter, supprimer et modifier le statut des autres

du coup pour :
-un invité : un bouton ajouter une empreinte
-un employé: rien
-un patron : un bouton supprimer une personne (et son empreinte), modifier le statut d'une personne (choix : invité, employé ou patron)

les données :
ID
surname
firstname
status
psw

les + si on a le temps :
-envoyer une notif au patron si une personne s'ajoute sur la base de donnée
